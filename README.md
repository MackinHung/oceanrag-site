# OceanRAG — 靜態網頁介紹

> 專為中小型企業設計的 AI 內部智慧體系統，零成本顧問進駐，100% 斷網私有化。

## 頁面

| 頁面 | 檔案 | 說明 |
|------|------|------|
| 首頁 | `index.html` | 產品介紹、功能亮點、CTA |
| 深入了解 | `docs.html` | 系統架構、技術規格、部署指南 |
| 開發者 | `developer.html` | 開發者資源與 API 文件 |
| 合作聊聊 | `contact.html` | 預約 POC 展示、產品諮詢 |
| 免責聲明 | `disclaimer.html` | 使用條款 |

## 技術

- 純靜態 HTML / CSS / JavaScript
- 響應式設計 (RWD)
- Google Fonts (Inter) + Font Awesome icons
- 無框架依賴，零建置步驟

## 部署

直接用任意靜態託管服務部署根目錄即可：

- **GitHub Pages**: Settings → Pages → Branch: `main` → `/` (root)
- **Cloudflare Pages**: 連接 repo，無需 build command
- **本地預覽**: 用任意 HTTP server 開啟，例如 `npx serve .`
